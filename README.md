Simple program based on the principles of geometric optics to compute

a) The phase function and the degree of linear polarization for unpolarized incident light scattered by a large spherical particle (refractive index m = 1.33, and size parameter $\frac{2\pi R}{\lambda} = 500$).

b) The phase function and the degree of linear polarization for unpolarized incident light scattered by a large hexagonal ice crystal under the random orientation condition in the simplified 2D case discussed in class (refractive index m = 1.31, the maximum dimension of the particle = $2a$, and size parameter $\frac{2\pi a}{\lambda} = 500$).
